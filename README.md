# check-reboot
## Description
detect reboot.

## Usage
```
usage: check-reboot [options]
    Description:
        Detect reboot.

    Options:
        -h  show help
        -w  set alert level WARNING (exit status 1). (default: not selected)
        -c  set alert level CRITICAL (exit status 2). (default: selected)
```
